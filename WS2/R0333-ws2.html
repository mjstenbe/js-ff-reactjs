Exercise 1 - Class based components with state

  1. Create class based component <HelloWorld />
  2. Define a state (class variable) called "greeting" for the component in constructor
  3. Output the greeting within some HTML-elements.
  4. Add styles, such as border and backgroundcolor etc.

Exercise 2 - Object state

  1. Create button component which renders a <HelloWorld />greeting to user when clicked
  2. Test drive your app
  3. Add another button labeled as "Translate", which calls for a method frenchify()
  4. Create a method frenchify() which changes the greeting text set in state variable (class variable) of HelloWorld-component to another language
  5. You can create other buttons as well, such as "englify" and "finnify".

Exercise 3 - Button and state

  1. Create a class based component called "<GenderButton />
  2. Button has a state called "gender", which keeps track whether this button is set to Male or Female. Default setting is Male.
  3. When the button is clicked, it will call for a method handleClick(). The method will change state-variable "gender" to either Female or Male, depending on the current value.
  4. Output a log message to the console displaying the button info
  5. Set the label of the button to the current value of Gender state.

Exercise 4 - Button as a component

  1. Create a class based component called "<Button />
  2. Button has a state called counter, which keeps track on the times it was clicked
  3. Button accepts its title (name shown on the screen) as props
    <Button title="First button"/>
    <Button title="Second button"/>
  4. When the button is clicked, it will call for a method increment(). The method will increment state-variable counter with one
  5. Output a log message to the console displaying the button and counter info

Exercise 5 - Modules

  1. Refactor the code produced during this workshop
  2. Copy the code for <Helloworld /> component into its own file. "Helloworld.js"
  3. At the end of the new file add the line: 
    export default HelloWorld
  4. Save the file under src/components/ -folder (you have to create the components-folder)
  5. Then import the module using import statement similar to the ones used before: 
    import './components/Helloworld.js';
  6. Test to see everything works.
  7. Then Do the same for other components in the workshop as well
